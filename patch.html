Numworks Unlock Patcher (beta version)

Got this page in French from https://www-fourier.univ-grenoble-alpes.fr/~parisse/numworks/patch.html, translated to English by DeepSeek.

Should work with Epsilon versions 20 to 25.

Select your internal flash dump and kernel slot A files to automatically apply patches that enable interoperability with your calculator's unlock. To obtain these dump files, go to the TI-Planet webUSB DFU site, retrieve internal flash and slot A.

If you prefer to work locally with dfu-utils (Linux/Mac), use the scripts (patchboot120.sh and patchkernel120.sh) or patchboot115.sh, with the dumps obtained by the following commands, for N0120:
/bin/rm -f epsilonkernel.bin
dfu-util -i 0 -a 0 -s 0x90000000:0x10000:force -U epsilonkernel.bin
/bin/rm -f epsiloninternal.bin
dfu-util -i 0 -a 0 -s 0x8000000:0x80000:force -U epsiloninternal.bin
On N0115, replace the last command with:
dfu-util -i 0 -a 0 -s 0x200000:0x10000:force -U epsiloninternal.bin
